## How to install virtual environment

1. You need python-venv package for this. Install it

```bash
# in ubuntu system
sudo apt install python-venv

# just use pip package
pip install venv
```

2. Now initialize virtual environment!

```bash
python -m venv .venv

source .venv/bin/activate
```

3. Install requirements.txt

```bash
pip install -r requirements.txt
```

4. Now you can do everything you want!
